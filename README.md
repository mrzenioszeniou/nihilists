## Install

Install:

```
cargo install --git https://github.com/mrzenioszeniou/nihilists
```

Uninstall:

```
cargo uninstall nihilists
```

## Usage

To start:

```
nihilists
```

* Move around using arrow keys.
* Press `+`/`-` to change the number of allotted nihilists.
* Press `Q` to exit.
